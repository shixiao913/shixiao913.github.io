Markdown 学习笔记 (基础篇)

Posted on 2020-02-05 Edited on 2020-02-12

通过使用来学习markdown。

本篇主要练习内容：

  • 字体变化和emoji
  • 列表
  • 链接和脚注
  • 标题、横线

参考资料: markdown-cheatsheet，markdown教程和Markdown 语法快速入门手册

第一部分 字体变化和emoji

Common text 普通文本

Italic 斜体字

Strikethrough text 删除线

Bold text 加粗

Bold italic text 斜体加粗

Underline 下划线

Emoji: :wink: :blush: :question:

第二部分 列表

list 普通列表：

豆瓣新片榜：

  • 《82年生的金智英》
  • 《 小丑 》
  • 《婚姻故事》
  • 《 爱尔兰人》
  • …

Numbered list 带序号的列表：

豆瓣新片榜

  1. 《82年生的金智英》
  2. 《 小丑 》
  3. 《婚姻故事》
  4. 《 爱尔兰人》
  5. …

Nested list 列表嵌套：

  1. 我喜欢的水果:
    • 西瓜
    • 榴莲
  2. 我喜欢的蔬菜
    • 生菜
    • 油麦菜

第三部分 链接和脚注

Web Link 网页链接：

Named Link and http://www.myblogaddr.cn or https://blablabla.com/

Title Link 链接至标题：

[比如连接至最开头的第一个标题](#Markdown 学习笔记 (基础篇))

Email 邮箱：

example@exmple.com

Footnote 脚注：

we need a [^footnote] here.
这里需要一个[^脚注]。
…

[^footnote]: some discription
[^脚注]: 一些描述.
